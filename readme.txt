Display contact cards of your site's users.

== Description ==

Display contact cards of your site's users with the ease of just inserting the [contact] shortcode.

== Installation ==

You can download and install Contact Card using the built in WordPress plugin installer. If you download Contact Card manually, make sure it is uploaded to "/wp-content/plugins/contact-card/".

Activate Contact Card in the "Plugins" admin panel using the "Activate" link. If you're using WordPress Multisite, you can choose to activate Contact Card network wide for full integration with all of your sites.
